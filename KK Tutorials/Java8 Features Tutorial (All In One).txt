-Default and Static methods in Interfaces
	It is not possible for an interface to provide default implementations for the methods in Object.Trying to do so will result in a compile error.
	IF any class in the hierarchy has a method with the same signature as Object class methods, then default methods become irrelevant.
	Normally static method in interface is used as Helper methods while default method used as default implementation for classes that implements that interface.
	Default methods can be overridden in implementing class while static cannot.
